Ce plugin interroge vianavigo.com pour vos donner les horaires de votre prochain train.

Auteur

Fabien0102

Version

1.0.0

Configuration

Afin que SARAH vous donne les informations qui vous correspondent, il faut lui donner quelques informations :

  • departure : L'arrêt de départ
  • departureCity : La ville de départ
  • departureCityCode : Le code postal de la ville de départ
  • departureExternalCode : Un code mistérieux de la ville de départ
  • arrival : L'arrêt d'arrivée
  • arrivalCity : La ville d'arrivée
  • arrivalCityCode : Le code postal de la ville d'arrivée
  • arrivalExternalCode : Un code mistérieux de la ville d'arrivée
  • trollTime : Temps en minutes où SARAH te dit que tu vas être à la bourre

Toutes ces informations se trouve dans la barre d'adresse quand vous faite un requête sur vianavigo (le site mobile, pour faciliter le scrapping)
